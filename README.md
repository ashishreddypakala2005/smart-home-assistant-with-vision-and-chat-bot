🏠 **Smart Home Assistant with Vision and Chatbot**
📌 **Description**
This project implements a Smart Home Assistant by integrating computer vision and a rule-based chatbot.
The system detects objects in an image using YOLO (OpenCV) and responds to user queries based on the detected objects.

🚀 **Features**

* Object detection using YOLO (COCO dataset)
* Detection of multiple objects in a scene
* Chatbot-based interaction
* Context-aware responses
* Continuous user interaction

🛠️ **Technologies Used**

* Python
* OpenCV
* YOLOv3
* NumPy

⚙️ **How It Works**

* Upload an image of a home environment
* Perform object detection using YOLO
* Extract detected object labels
* Ask questions using chatbot
* Get responses based on detected objects

▶️ **Example Queries**

* What do you see?
* Is there a chair?
* How many objects are present?
* Is anyone in the room?

📸 **Output Screenshots**

* Input Image
* Detected Output
* Chatbot Response

📊 **Results**
The system successfully detects objects such as chair, couch, TV, and plant, and responds intelligently to user queries using chatbot logic.

🎯 **Applications**

* Smart home monitoring
* Security systems
* Assistance for elderly and visually impaired
* Human-computer interaction

🎯 **Conclusion**
This project demonstrates the integration of computer vision and conversational AI to create an intelligent smart home assistant.
